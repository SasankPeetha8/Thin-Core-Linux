# Thin-Core-Linux
Thin Core Linux is a customised Linux distribution built on top of Rocky Linux 10, chosen for its enterprise-grade stability and thoroughly tested package ecosystem.
This distro is specifically optimised for lightweight performance, custom kernel configurations, and a minimal software footprint tailored to selected hardware.

**Default software included:**
1. KDE Plasma Workspace
2. PowerShell
3. ZSH
4. .NET SDK
5. Rust Toolchain
6. VS Code
7. Vim
8. Thunderbird
9. Vivaldi Browser
10. NVIDIA Graphics Driver

Thin Core Linux project provides me a sleek, efficient, and developer-friendly environment with carefully chosen tools for productivity, coding, and daily use.
